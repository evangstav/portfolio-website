# Downloads Folder

Place downloadable files here:

## Suggested Files

### Biography Documents
- `biography.pdf` - Full biography in PDF format
- `biography-short.pdf` - Short biography (1-2 paragraphs) for programs

### Press Kit
- `press-photos.zip` - High-resolution photos for press use
- `press-kit.pdf` - Complete press kit with bio, photos, and repertoire

### Other Documents
- `repertoire.pdf` - Complete repertoire list
- `cv.pdf` - Curriculum Vitae

## Guidelines

- PDF files should be print-quality
- Photos should be minimum 300 DPI for print use
- Include both color and B&W versions of key photos
- Keep file sizes reasonable for web download
